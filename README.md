# MessageVault

MessageVault is a Flask-based web application that helps users save, organize, search, and manage useful messages collected from social media platforms.

## Features

- Add Notes
- Edit Notes
- Delete Notes
- Search Notes
- Read Notes
- Source Tracking
- Important Notes
- Dashboard Statistics
- SQLite Database
- Logging

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- Bootstrap
